🛠 Google OAuth Setup Guide

Step 1 — Create Google Cloud Project

Go to Google Cloud Console. Create or select a project.

Step 2 — Enable Gmail API

Open Gmail API → Click Enable.

Step 3 — OAuth Consent Screen

APIs & Services → OAuth consent screen. External, add scope https://mail.google.com/, add Test users.

Step 4 — Create OAuth Client ID

Credentials → Create → OAuth client ID. Type: Web application. Add origins: https://yourusername.github.io and http://localhost. Copy Client ID.

Step 5 — Paste & Sign In

Paste Client ID on sign-in screen, click Sign in with Google.
