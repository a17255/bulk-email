🛠 Google OAuth Setup Guide

Step 1 — Create a Google Cloud Project

Go to Google Cloud Console.
Create a new project (or select an existing one).

Step 2 — Enable Gmail API

Open Gmail API in the API Library.
Click Enable.

Step 3 — Configure OAuth Consent Screen

Go to APIs & Services → OAuth consent screen.
Select External, fill in the app name, support email.
Add scope: https://mail.google.com/
Add yourself as a Test user (and any other users, up to 100).

Step 4 — Create OAuth Client ID

Go to APIs & Services → Credentials.
Click Create Credentials → OAuth client ID.
Application type: Web application.
Authorized JavaScript origins: add your GitHub Pages URL
(e.g., https://yourusername.github.io).
For local testing, also add http://localhost.
Copy the Client ID.

Step 5 — Paste Client ID

Paste your Client ID into the field on the sign-in screen.
Click Save, then Sign in with Google.
